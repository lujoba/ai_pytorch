## Set Up

1. Check out the code;
2. Install requirements:
    ```
    pipenv install
    ```
3. Use as CLI, as the example below, or call the class in your project:
    ```
   pipenv python -m stack_images.py folder/of/input/images path/of/output/images --method ORB 
    ```
